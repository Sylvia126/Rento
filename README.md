# 🏠 Rento — Property Rental Web Application

![Rento Banner](./Rento_Updated/Final%20Project/rento.png)

> A modern property rental platform built as a Human-Computer Interaction (HCI) project with a focus on usability, accessibility, and responsive design.

---

## 📖 Overview

**Rento** is a front-end web application that simulates a real-world rental platform for discovering and booking properties.

The project was designed to provide users with a smooth and intuitive experience while browsing rental properties, managing bookings, viewing locations on maps, and saving favorite listings.

The system emphasizes:

* User-centered design
* Accessibility and usability
* Responsive layouts
* Clean and modern UI/UX
* Interactive user experience

---

## ✨ Features

### 🏡 Property Browsing

* Browse rental listings with images and property details
* View pricing, ratings, and availability
* Responsive card-based layout

### 📍 Interactive Map

* Display property locations on a map
* Explore rentals geographically
* Improve navigation and property discovery

### ❤️ Favorites System

* Save preferred properties
* Access saved listings anytime

### 📅 Booking Management

* Book available rental properties
* Track and manage bookings

### 📊 User Dashboard

* Personalized dashboard for users
* View rental activity and property information

### 🔐 Authentication Pages

* Login and signup interfaces
* Client-side authentication handling

### 🌙 Dark Mode Support

* Toggle between light and dark themes
* Improved accessibility and user comfort

### 📱 Fully Responsive Design

* Optimized for desktop, tablet, and mobile devices
