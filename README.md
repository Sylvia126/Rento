Rento is a modern full-stack rental platform that allows users to browse, list, and manage rental properties/items in a simple and efficient way. It is designed to connect renters with owners through a clean and responsive web interface.

---

## ✨ Features

- 🔍 Browse rental listings easily  
- 🏡 View detailed property/item pages  
- 👤 User authentication (login/register)  
- 📅 Booking and rental management system  
- 💬 Communication between users (chat/contact)  
- ⭐ Ratings & reviews system (if implemented)  
- 📱 Fully responsive UI for mobile and desktop  
- ⚡ Fast and optimized performance
